# DESCRIPTION

- Web application manipulating the parallax design technique.

## STACK

- HTML5 & CSS3

## INSTALLATION

- Clone the project

```bash
  git clone https://github.com/Roulioo/parallax.git
```

- Go to the project directory

```bash
  cd parallax
```

## RUNNING

- Install dependencies

```bash
  npm install
```

- Run the project

```bash
  npm run start
```

## URL

- http://127.0.0.1:8080/
